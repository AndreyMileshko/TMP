# TMP Autonomous Cursor Development

Этот комплект переводит разработку TOP Manufacturing Platform в управляемый автономный режим.

Пользователь не подготавливает отдельный prompt для каждой задачи и не переносит контекст между чатами.
Cursor самостоятельно:

1. определяет текущий Stage;
2. строит или уточняет очередь маленьких задач;
3. выбирает следующую задачу со статусом `READY`;
4. загружает только разрешённый контекст;
5. выполняет Question Gate;
6. реализует задачу;
7. запускает обязательные проверки;
8. выполняет самопроверку;
9. обновляет контрольные документы;
10. переходит к следующей задаче либо останавливается на блокере.

Главный файл для запуска:

```text
docs/development-control/RUN-DEVELOPMENT.md
```

Главный файл для наблюдения:

```text
docs/development-control/STATUS.md
```
