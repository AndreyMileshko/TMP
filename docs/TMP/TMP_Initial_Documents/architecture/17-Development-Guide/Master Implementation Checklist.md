**Document ID:** TMP-IMP-001  
**Status:** Active  
**Version:** 1.0

---

# Назначение

Данный документ является главным чек-листом реализации TOP Manufacturing Platform.

Он используется на протяжении всей разработки и является единственным источником информации о текущем состоянии реализации проекта.

Каждый пункт может быть отмечен выполненным только после полного соответствия критериям завершения (Definition of Done).

---

# Правила работы

1. Реализация ведётся строго последовательно.
2. Новый этап начинается только после завершения предыдущего.
3. Cursor AI работает только в рамках текущего этапа.
4. Запрещено выполнять задачи будущих этапов.
5. После завершения каждой задачи обязательно обновляются:
   - данный чек-лист;
   - журнал реализации;
   - техническая документация (при необходимости).
6. Любое изменение архитектуры требует обновления соответствующей спецификации до продолжения разработки.
