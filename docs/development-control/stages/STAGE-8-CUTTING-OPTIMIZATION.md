# Stage 8 Manifest — Cutting Optimization

## Source

Cutting Optimization Specification, Production contracts and approved algorithm requirements.

## Planning domains

1. input model and validation;
2. material/profile model;
3. constraints;
4. deterministic optimization API;
5. baseline algorithm;
6. optimization metrics;
7. cutting map domain;
8. persistence/versioning if specified;
9. production integration;
10. visualization;
11. algorithm correctness tests;
12. performance tests with approved datasets.

## Exit criteria

Optimization is deterministic for fixed inputs, produces valid cutting maps, respects constraints and integrates through public APIs.
